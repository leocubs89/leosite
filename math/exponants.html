What is an exponent?

An exponent decides how many times you multiply a number or expression by itself.

Example 1: 2^2 = 2 ∙2 = 4   Example 2: 2^3 = 2 ∙ 2 ∙ 2 = 8

As you can see in the first example the exponent is 2. This means we multiply by 2, twice!

In the second example the exponent is 3. This means we multiply by 2, three times.

basic laws of exponants

the three basic laws of expoanants that we will learn are as follows

  1. x^1 = x
  2. x^0 = 1
  3. x^-1 = 1/x

Now lets take a closer look at these laws.

x^1 = x

Using our examples from above what do you think the answer would be to 2^1?

If you follow the pattern above, you will get: 2^1 = 2 = 2

Easy enough, right?(:

x^0 = 1

In this scenario would be having an exponent of 0. For example: 2^0=1.

Now this might not work using the patterns from above but what if we rewrite our first example as: 2^2 = 1 ∙ 2 ∙2 = 4

Now following this pattern, 2^0=1 Makes sense. In fact any number or expression with an exponent of 0 will always be 1.

x^-1 = 1/x

If we have a negative exponent then there is only one thing we need to do differently and that is to put the number or expression in the denominator.

for example: 2^(-2) =1/(2^2) from here we can use the same patterns used above to find 2^(-2) = 1/(2^2) = 1/(1 ∙ 2 ∙ 2) = 1/4

fractional exponants

fractional exponents are another story. If you have an exponent with a number in the denominator like here: 2^(1/2) then this the 2nd root of 2: 2^(1/2)= √2.

if we have 2^(1/3), then we can see the 3 in the denominator and know this is the 3rd root of 2: 2^(1/3)= ∛2

this goes on and on for all numbers.

Using what we have learned so far

Putting them all together we can get something crazy like: 4^(-3/2). This can seem like a lot at first but using what we know from above, we can solve it!

The best part about the rules from above is that we can apply them in any order. Lets deal with the negative sign first.

This means we must move the expression into the denominator like so: 4^(-3/2)= 1/(4^(3/2))

Next we can deal with the 2 in the denominator in our exponent.

This means we take the 2nd root of our expression giving us: 4^(-3/2) = 1/(4^(3/2)) =1/(√(4)^3)

Solving √(4)=2: we get 4^(-3/2 )= 1/(4^(3/2)) = 1/(√(4)^3) = 1/(2^3)

Last step is to deal with our exponent of 3 using the first example we saw: where 2^3= 1∙ 2 ∙2 ∙2=8

Finally giving us 4^(-3/2)= 1⁄4^(3/2) =1⁄〖√(2&4)〗^3 = 1⁄2^3 =1⁄( 1∙ 2 ∙2 ∙2=1⁄8)

I encourage you to try this problem by doing the steps in a different order and see if you get the same answer. Hint: you will!

more advanced laws of exponants

the three more advanced laws of expoanants that we will learn are as follows

  1. (x^m)(x^n) = x^(m+n)
  2. (x^m)/(x^n) = x^(m-n)
  3. (xy)^n = (x^n)(y^n)
  4. (x/y)^n = (x^n)/(y^n)

Now lets take a closer look at these laws.

(x^m)(x^n) = x^(m+n)

Here we are multiplying the same number but with different exponents. For example (2^2)∙(2^3)

Plugging in our equation we get: (2^2)∙(2^3)=2^5

Now we simply solve using our basic rules to find 2^5=1∙2∙2∙2∙2∙2=32

(x^m)/(x^n) = x^(m-n)

Here we are dividing the same number but with different exponents. For example (2^2)/(2^3)

Plugging in our equation we get: (2^2)/(2^3)=2^-1

Again, we have gotten to a point where we can use our basic rules to solve this to be: 1/2

(xy)^n = (x^n)(y^n)

here our base numbers are not the same so we must distribute the exponant among them.

lets use the example (2∙1)^3

simply plugging this into our equation we get (2∙1)^3 = (2^3)∙(1^3)

then solve using our basic laws: (2∙1)^3 = (2^3)∙(1^3) = (2∙2∙2)∙(1∙1∙1) = 8

(x/y)^n = (x^n)/(y^n)

following the same logic as the pervious rule we must distribute the exponant among our terms.

lets use the example (1/2)^3

simply plugging this into our equation we get (1/2)^3 = (1^3)/(2^3)

then solve using our basic laws: (2∙1)^3 = (2^3)/(1^3) = (1∙1∙1)/(2∙2∙2) = 1/8
