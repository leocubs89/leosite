HTML Lesson 1

HTML Set up

       
              <!DOCTYPE html>
              <html>
              <head>
                  <title>Page Title</title>
              </head>
              <body>
                  
              </body>
              </html>

Above is the basic set up for a HTML document. Most of what you can see are called Tags and we will be taking a closer look at what they do below.

HTML Tags

              <!DOCTYPE html>
              <html>

              </html>

On the first line we can see the HTML document type declaration. This is not technically a tag, however, it is similar. all HTML documants must have this.

Next we have the Opening Tag for our HTML Tag. This is then followed by the Closing Tag. The closing can be distinguished by the forward slash '/'.

Most tags will be made up of an opening tag and a closing tag (kinda confusing sometimes because 1 tag is made up of 2 tags but please just work with me). however, there are some exceptions to every rule, such as the image tag but we will get to that later.

More Tags

       
              <!DOCTYPE html>
              <html>
              <head>
                  <title>Page Title</title>
              </head>
              <body>
                  
              </body>
              </html>

Back to where we began. The rest of the tags have been returned. Lets take a closer at the meaning of each tag and how they are postioned.

The first new tag we see is the header opening tag. the header is being opened after the HTML opening tag but before the clasing html tag. This means the header tag is inside of the html tag. This then means the header tag must close before the html tag.

Next we see the title tag inside of the header tag, which we of course know means it must close before the header tag. Inside of the title tag we see the text "Page Title", this is of course to represent the title of the page you are creating. I think a good title for this page would be "HTML Lesson 1", so lets change it 

<title>HTML Lesson 1</title>

Rock and roll! we also closed the title tag on the same line. Then above you see we closed the header tag. Now something imporant about the header tag is that nothing inside of the header is immediatly visable on the webpage. This is where the body tag comes into play.

body Tags

              <body>
                This is where the fun begins
              </body>
              </html>

Here we have the body tag and I have added some text inside. This text would show up immediatly on the web page since it is inside of the body tag. We then of course close the body tag before the html tag and our first web page is finished. Try making a file using what we learned here and open it with your web browser.

Before you go: p tags

              <body>
                <p>
                  This is where the fun begins
                </p>
              </body>
              </html>

Since I believe in you, I am gonna tell you about one more tag. the p or the paragraph tag. It is good practice to put your text inside the p tag, as I have above. We will be looking at more tags and ways to break up our information in future lessons.
